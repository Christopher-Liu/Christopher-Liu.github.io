Bike Rental Time Series Analysis and Forecasting

An end-to-end analysis and of bicycle rental data from Capital Bikeshare, a bicycle sharing company operating in the Washington DC metro area, that goes all the way back to 2013. Since the full data set was larger than the available amount of memory, data aggregation and transformation with regular in-memory tools was not possible, so PostgreSQL was utilized for loading and aggregating the data in chunks. Exponential smoothing and ARIMA time series methods were then applied to produce accurate models that fit the data to within 8% error and provided robust forecasts of expected bicycle rentals in the coming year. All of the data exploration and analysis was done with Python and its associated data analysis libraries (pandas, NumPy, matplotlib), and the time series modeling was done with the statsmodels library.

Learn PCA Tutorial - R Shiny App

Interactive tutorial for learning the theory behind PCA (Principal Components Analysis) and how to apply it in R. PCA is a popular technique used in data science/machine learning as well as research domains, such as neuroimaging (Li et al., 2008, Viviani et al., 2004), for reducing the dimensionality of data without significant loss of information. PCA is particularly useful in "n << p" problems, where the amount of variables far outnumbers the sample size. The tutorial was made with R Markdown and R Shiny.
