Nested Cross Validation - R Package

R package for performing nested cross validation in order to do simultaneous model training and model performance assessment. In certain situations, "regular" cross validation potentially leads to optimistically biased estimates of a model's true test set performances (Varma et al., 2006), hence the need to adjustments to provide for more accurate, unbiased estimates.

Learn PCA Tutorial - R Shiny App

Interactive tutorial for learning the theory behind PCA (Principal Components Analysis) and how to apply it in R. PCA is a popular technique used in data science/machine learning as well as research domains, such as neuroimaging (Li et al., 2008, Viviani et al., 2004), for reducing the dimensionality of data without significant loss of information. PCA is particularly useful in "n << p" problems, where the amount of variables far outnumbers the sample size. The tutorial was made with R Markdown and R Shiny.
